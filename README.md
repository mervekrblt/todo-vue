# todo-vue

This is a basic to-do project building with vue.js.

Technologies that I used
- Vue.js
- Vuex
- Axios
- Fontawesome

## How to Use
Clone the repository:

```
git clone https://github.com/mervekrblt/todo-vue.git
```
    
Install all dependies and run the project

```
npm install

npm run serve
```
